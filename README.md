# Bili2Gemini 视频提炼助手

Bili2Gemini 是一款直达痛点的视频内容提炼工具。一键提取 Bilibili 与 YouTube 的**字幕**或**精华评论**，并**自动建立桥梁**，跳转至 Gemini 进行 AI 智能总结。

## ✨ 核心亮点
- **桥接式提炼 (Bridge to Gemini)**：点击按钮后，不仅会提取文本，还会**自动打通到 Gemini 的链路**，光速跳转、粘贴并执行您的专属 Prompt。
- **一键提取字幕**：支持 YouTube 和 B站视频，自动优先匹配中文字幕轨道。
- **精华热评直通车**：支持 B站，一秒内拉取最高赞的 1000 条热评（含子回复），丢给 AI 分析社区风向。
- **极简明示 UI**：最纯粹的胶囊宽屏按钮（字幕 ➜ Gemini、热评 ➜ Gemini），一眼便知核心功能。
- **自定义提示词 (Prompt)**：内置选项设置页面，支持分别为“字幕”和“评论”配置不同的 AI 指令。
- **极致 Token 优化**：针对 AI 接口过滤一切无效空字符和 Emoji。

## 🚀 使用说明
1. **安装扩展**：
   - 在浏览器扩展管理页面开启“开发者模式”。
   - 点击“加载已解压的扩展程序”，选择本项目所在的 `bili-suite` 文件夹。
2. **提炼视频**：
   - 打开任意 B站视频（支持字幕+评论）或 YouTube 视频（支持字幕）。
   - 点击右侧悬浮按钮：
     - **蓝色/紫色按钮**：提取完整字幕。
     - **粉色按钮**：提取精华评论。
3. **个性化配置**：
   - 点击**灰色齿轮按钮**即可直达设置页面，修改点击提取后自动附带的总结指令。

## 📂 目录结构
- `manifest.json`: 扩展配置文件。
- `content.js`: 核心逻辑脚本（包含提取、清理与跳转逻辑）。
- `style.css`: 悬浮按钮组与 Toast 提示样式。
- `options.html/js`: 提示词配置页面及其交互逻辑。
- `icons/`: 插件图标资源。
