# Bili-Q-TLDR Pro

Bili-Q-TLDR Pro 是一款全能的视频提炼助手，专为 B站 (Bilibili) 和 YouTube 设计。它可以一键提取视频字幕或精华评论，并自动跳转至 Gemini 进行智能总结。

## ✨ 主要功能
- **一键提取字幕**：支持 YouTube 和 B站视频，自动优先匹配中文字幕轨道。
- **精华评论提炼**：支持 B站视频，顺序抓取前 1000 条热评（含子评论），并自动按点赞数执行本地严格降序排列。
- **自定义提示词 (Prompt)**：内置选项设置页面，支持分别为“字幕”和“评论”配置不同的总结 Prompt。
- **Gemini 自动粘贴**：提取成功后自动复制到剪贴板，跳转至 Gemini 页面并完成粘贴与提交。
- **极简 UI 设计**：三个透明感悬浮按钮（字幕、评论、设置），不打扰网页原本布局。
- **优化性能**：移除所有 Emoji 字符以节省 Token，具备防风控重试机制。

## 🚀 使用说明
1. **安装扩展**：
   - 在浏览器扩展管理页面开启“开发者模式”。
   - 点击“加载已解压的扩展程序”，选择本项目所在的 `bili-suite` 文件夹。
2. **提炼视频**：
   - 打开任意 B站视频（支持字幕+评论）或 YouTube 视频（支持字幕）。
   - 点击右侧悬浮按钮：
     - **蓝色/紫色按钮**：提取完整字幕。
     - **粉色按钮**：提取精华评论。
3. **个性化配置**：
   - 点击**灰色齿轮按钮**即可直达设置页面，修改点击提取后自动附带的总结指令。

## 📂 目录结构
- `manifest.json`: 扩展配置文件。
- `content.js`: 核心逻辑脚本（包含提取、清理与跳转逻辑）。
- `style.css`: 悬浮按钮组与 Toast 提示样式。
- `options.html/js`: 提示词配置页面及其交互逻辑。
- `icons/`: 插件图标资源。
